# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][covenant] as its
code of conduct. Please read the full text at the link below.

Concerns or reports about contributor behavior in this project should be sent
privately to **sebastienmelki@gmail.com**. Reports are handled
confidentially; the project maintainer responds within a reasonable window
(typically a few business days) and follows the enforcement guidelines from
the Contributor Covenant.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
